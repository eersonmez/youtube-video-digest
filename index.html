  • It versions the activity between commits, including edits, prompts, and agent conversations.
  • Work trees and agent sessions become collaborative artifacts that teammates can join later.
  • Fine-grained deltas allow code to be traced across time, movement, deletion, and refactoring.
  • Sobo acknowledges privacy risks and discusses controls for sharing only selected history.
