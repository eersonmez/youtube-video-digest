The new ChatGPT application allows users to control websites and desktop applications to complete various tasks.

What This Video Is About

This video introduces a significant update to the ChatGPT application that allows users to move beyond simple text generation into performing real, actionable work by controlling existing websites and desktop applications. The core concept is extending ChatGPT's capabilities from a conversational interface to an agent capable of interacting with digital environments as if it were a user.

The functionality is delivered through several mechanisms: an integrated in-app browser for web tasks, a Chrome extension for leveraging existing browser sessions, and a "Computer Use" feature that enables control over native desktop applications on both Windows and macOS. This shift allows users to automate workflows such as data entry or generating reports directly within the context of their daily software usage.

Detailed Walkthrough

The video outlines several ways the new ChatGPT application can be utilized for task completion. One primary method involves using the in-app browser [0:20]. For instance, a user might instruct ChatGPT to analyze recent posts from a community forum to identify top issues causing user frustration over the past month. The in-app browser handles this by scanning the specified forum and compiling a friction report [0:20]. Furthermore, this integrated browser includes an annotation tool, which allows users to offer targeted feedback or ask specific questions on a particular page without needing to write lengthy instructions [0:28].

If the required interaction needs to happen within the user's currently open browsing session rather than in the dedicated in-app browser, the Chrome extension provides a connection point [0:36]. This allows ChatGPT to interact with tabs that are already open or create new ones discreetly in the background, which is beneficial for tasks like filing expense reports or managing long data entry workflows [0:42, 0:47].

For interactions that occur outside of web browsers—specifically within native desktop applications on Windows or macOS—the "Computer Use" feature is employed [0:52]. On a macOS environment, this capability is enhanced because ChatGPT operates with its own cursor. This allows it to function independently in the background, enabling users to continue working on other tasks while workflows like data entry or expense report filing are being completed by the AI [1:00].
