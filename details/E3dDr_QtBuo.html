ChatGPT's new capabilities for working directly with documents, spreadsheets, and slide decks.

OpenAI · 2026-07-23T14:17:31+00:00 · Work with docs, sheets, and slides in ChatGPT

What This Video Is About

This video demonstrates the expanded capabilities of ChatGPT, specifically its ability to interact directly with various file types such as documents, spreadsheets, and slide decks. The central theme is how this integration streamlines complex workflows for tasks like launch planning, performance analysis, and presentation generation. By connecting ChatGPT to existing project materials—such as decisions scattered across Slack or media notes—users can consolidate disparate information into a single, coherent document [0:11].

The tool allows users to perform granular edits within these documents while preserving the integrity of other sections, ensuring that updates are precise rather than wholesale replacements. Furthermore, it extends beyond simple content generation by enabling data analysis on past performance metrics and automating the creation of presentation materials based on established styles. This capability significantly reduces manual effort in project management and reporting across teams.

Detailed Walkthrough

The demonstration begins with a scenario where a team is preparing for a launch [0:04]. The current launch plan's latest decisions are fragmented, residing in Slack and various media notes [0:11]. The user instructs ChatGPT to consolidate this information into one coherent launch plan document [0:17], which results in the tool updating the decisions, owners, and overall plan within the existing document structure [0:20]. A key feature highlighted is the ability to perform targeted edits; for instance, a user can select a specific section and request a focus change while leaving all other parts of the document untouched [0:23]. For team collaboration, the content can be exported directly to Google Docs [0:28].

Moving to performance analysis, the video shows how ChatGPT can analyze historical video data. The user prompts it to compare metrics such as viewer retention based on video length and engagement levels categorized by format [0:31]. The output of this analysis is structured into a spreadsheet for easy review [0:36]. Finally, the process covers presentation generation. By providing an existing launch deck as a reference point, ChatGPT can generate a new presentation that adheres strictly to the structure and style of the provided example [0:48]. To ensure future consistency, the user utilizes the Template Creator skill to convert this newly generated file into a reusable template [0:56]. The video concludes by noting that these capabilities allow users to point ChatGPT toward necessary files, specify required actions, and receive polished documents, spreadsheets, and decks ready for review and sharing more quickly [1:14].
