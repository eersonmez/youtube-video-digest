Advanced .NET testing techniques to improve test effectiveness and coverage.

NDC Conferences · 2026-07-08T08:54:13+00:00

.NET Testing Techniques You Didn’t Know You Needed - Dante De Ruwe - NDC Copenhagen 2026

The presentation by Dante De Ruwe introduces several advanced testing methodologies beyond standard example-based testing. He discusses the value of failing tests as a safety net, contrasting it with incomplete tests that pass despite code changes. Key techniques covered include randomized testing, which guards against bias by generating random inputs rather than relying on predefined edge cases. Furthermore, property based testing, exemplified using FsCheck and the Mars Rover kata, allows developers to assert general properties of the code. Other crucial topics addressed are snapshot testing for tracking complex data changes, and various metrics like branch coverage and mutation score to assess test quality. The speaker advises that while simple testing is a good start, these advanced tools offer deeper insights into code correctness.

  • Randomized testing helps guard against bias by generating random inputs instead of relying solely on predefined cases.
  • Property based testing allows assertions about general properties of the code rather than specific examples.
  • Snapshot testing is useful for tracking changes in complex outputs like XML or API contracts.
  • Mutation score measures if tests are actually effective at catching introduced bugs, going beyond simple line coverage.
  • Flaky tests should be addressed by using time abstractions and increasing sample sizes to ensure robustness.
