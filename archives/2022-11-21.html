Implementing a WaveNet-like architecture for character-level language modeling.

Andrej Karpathy · 2022-11-21T00:32:48+00:00

Building makemore Part 5: Building a WaveNet

This lecture details the progression from a simple multi-layer perceptron (MLP) to a more complex, hierarchical model resembling WaveNet for predicting characters in a sequence. The implementation involves replacing basic layers with custom modules like `Embedding` and `FlattenConsecutive`, which allow for processing groups of consecutive input characters in parallel rather than flattening the entire context. Initial performance showed improvement by simply increasing the context length from three to eight characters, lowering validation loss from 2.10 to 2.02. Further refinement involved fixing a bug within BatchNorm layers when handling single-example batches and restructuring the network into a deeper, hierarchical design. After significant tuning, the model achieved a validation performance of 1.993 by increasing embedding size and hidden units, demonstrating improved stability in statistics estimation.
