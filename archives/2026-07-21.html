Plugins in ChatGPT

OpenAI’s video demonstrates how Plugins in ChatGPT connect services such as Gmail, Slack, Google Drive, and Canva to help individuals and teams complete work. In the plugins directory, users can link an app and add skills that provide a playbook for tasks such as structuring reports, preparing for meetings, or guiding repeatable workflows. The demonstration uses Canva to create a 1950s retro-futuristic flyer for a Student Science Fair, which the user can then edit in Canva. It also shows a sales plugin preparing for an Astor Peak review by gathering a relevant email, the latest proposal from Drive, and recent Slack decisions, then producing an agenda, risk breakdown, and open questions. ChatGPT can save the resulting brief as a new document and share it with the team, illustrating a workflow of gathering context, doing work, and producing an output inside ChatGPT.

  • Plugins connect ChatGPT to Gmail, Slack, Google Drive, Canva, and other tools.

I Built the Future of Learning for Developers

Nick Chapsas presents Dometrain as an AI-first learning platform he has substantially rebuilt for developers. It combines hands-on browser-based courses with guided instructions, a Monaco editor, autocomplete, runnable tests, system-design diagram exercises, and an in-browser Git terminal. Learners can ask an agent for hints, request a confirmed solution, inspect code changes, see tests run, and receive line-by-line or diagram-step explanations. The agent also supports video-course recaps, quizzes, topic searches with timestamps, code extraction, screen explanations, personalized learning paths, public profiles, achievements, and skill assessments that generate study plans. Dometrain offers free introductory C# and system-design hands-on courses, while its Light subscription includes hands-on content, interview questions, and system-design courses. Pro adds video courses and MCP access. Chapsas says a separate video will cover the platform’s architecture, agents, costs, and security.
